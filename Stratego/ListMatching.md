List Matching {#list-matching .twikiTopicTitle}
=============

::: {.twikiWebTitle}
Stratego \-- Strategies for Program Transformation
:::

In [ListMatching](ListMatching){.twikiLink} you can use list variables
inside a match against a list pattern. List variables are variables
followed by a `*`. List variables represent list content: some terms
that can be put in a list.

The current implementation in
[StrategoRelease09](StrategoRelease09){.twikiLink} just supports simple
matching like `[t*]` and `[t,t*]`. This is still an experimental
feature; full support to follow.

\-- [MartinBravenboer](../Main/MartinBravenboer){.twikiLink} - 25 Jan
2003
