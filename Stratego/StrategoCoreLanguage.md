Stratego Core Language {#stratego-core-language .twikiTopicTitle}
======================

::: {.twikiWebTitle}
Stratego \-- Strategies for Program Transformation
:::

The syntax definition of Stratego is organized by syntactic category
(Strategy, Term, etc.). This makes it hard to distinghuish syntactic
sugar from core constructs. Refactor the syntax definition to make this
distinction clear. This is also useful for static checks on the
components of the compiler, i.e., to verify that only a restricted set
of constructs is used at some point in the compilation process.

\-- [EelcoVisser](../Main/EelcoVisser){.twikiLink} - 30 Jul 2003\
[]{#TopicEnd}
