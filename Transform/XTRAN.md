XTRAN {#xtran .twikiTopicTitle}
=====

::: {.twikiWebTitle}
Program-Transformation.Org: The Program Transformation Wiki
:::

Quoting from <http://www.xtran-llc.com/xtran.html> :

[XTRAN](XTRAN){.twikiLink} is a proprietary expert system for symbolic
manipulation of computer languages, including:

-   Assemblers
-   Third generation languages such as C and Pascal
-   Fourth generation languages
-   Markup languages such as [HTML](HTML){.twikiLink} and
    [XML](XML){.twikiLink}
-   Database languages
-   Scripting languages
-   Special-purpose languages

Through manipulation of its internal representation of these languages,
typically specified through its powerful rules language,
[XTRAN](XTRAN){.twikiLink} provides:

-   Analysis
-   Translation
-   Re-engineering
-   Standardization
-   Code generation

In addition, you can use [XTRAN](XTRAN){.twikiLink} to perform powerful
text processing. This is an example of how [XTRAN](XTRAN){.twikiLink}\'s
power can be applied to essentially any problem domain by adding a set
of primitives for that domain to [XTRAN](XTRAN){.twikiLink}\'s rules
language.

[XTRAN](XTRAN){.twikiLink} was invented, designed, implemented, and
documented by our President and founder, [Stephen
Heffner](StephenHeffner){.twikiLink}.

------------------------------------------------------------------------

The language/system is propietary, and it is not clear from the website
what kind of technology is being used.

\-- [EelcoVisser](../Main/EelcoVisser){.twikiLink} - 28 Oct 2002\
