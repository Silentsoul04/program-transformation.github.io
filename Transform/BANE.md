BANE {#bane .twikiTopicTitle}
====

::: {.twikiWebTitle}
Program-Transformation.Org: The Program Transformation Wiki
:::

**Homepage:** <http://www.cs.berkeley.edu/Research/Aiken/bane.html>

*The Berkeley ANalysis Engine ([BANE](BANE){.twikiLink}) is a toolkit
for constructing program analyses such as dataflow and type inference
systems.*

The toolkit has been developed by [AlexAiken](AlexAiken){.twikiLink} and
students.
