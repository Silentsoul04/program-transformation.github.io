::: {.twikiWebTitle}
Sdf
:::

[here](http://bugzilla.sen.cwi.nl:8080/index.cgi).

Use the components \'sglr\' and \'pgen\' to report bugs in the core SDF
system, \'sdf-meta\' for IDE bugs and \'unknown\' if you just don\'t
know.\
