::: {.twikiWebTitle}
Spoofax
:::

install it here:
<http://www.eclipse.org/downloads/packages/eclipse-classic-372/indigosr2>

Inside eclipse, install the unstable version of Spoofax using the
following update site: <http://spoofax.org/update/unstable>\
