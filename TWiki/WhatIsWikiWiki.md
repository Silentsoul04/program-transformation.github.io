[]{#FAQ}[]{#FAQ_} FAQ:
----------------------

So what is this WikiWiki thing exactly?

[]{#Answer}[]{#Answer_} Answer:
-------------------------------

A set of pages of information that are open and free for anyone to edit
as they wish. They are stored in a server and managed using some
software. The system creates cross-reference hyperlinks between pages
automatically. See [WelcomeGuest](WelcomeGuest){.twikiLink} and
[TWikiSite](TWikiSite){.twikiLink} for more information.
